## 📌 About The Project

This is my personal portfolio website built from scratch using **HTML, CSS, and JavaScript**.

The purpose of this project is to create a professional online presence where I can showcase my frontend development skills, featured projects, and continuous learning journey.

- 🌙 Dark / Light mode toggle

- 🎨 Modern UI with animations and hover effects
